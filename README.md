# ApiMonkey
ApiMonkey is a Rails 5 engine designed to make data-driven API
development a breeze by eliminating the boilerplate necessary for a
standard Rails REST API.

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'api_monkey'
```

And then execute:
```bash
$ bundle
```
